# System rezerwacji sal konferencyjnych

## Requirements

- Python 3.12
- npm

## Setup

```shell
git clone github.com/rikardoricz/conference-room-booking
cd conference-room-booking
```

### Backend

```shell
cd backend
python3 -m venv .venv
. .venv/bin/activate
pip install -r requirements.txt
flask run
```

### Frontend

```shell
cd frontend
npm install
npx expo start
```
